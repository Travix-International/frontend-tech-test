# Introduction

Small Responsive app to manage your own TODOs. 

# Install

## Windows 
* run `build.cmd`
* go to artifacts and open the freshly built folder
* run `node server.js [--port your_port]`
* access the browser on specified port (default 9001)

## Linux Based Systems
* run `make`
* go to artifacts and open the freshly built folder
* run `node server.js [--port your_port]`
* access the browser on specified port (default 9001)
