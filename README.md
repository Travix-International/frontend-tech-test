#Scripts

## Server
### `yarn server`
Runs server using nodemon.

## Client
### `yarn install`

Installs the app.

### `yarn start`

Runs the app in the development mode.

### `yarn test`

Runs the test runner.

### `yarn build`

Builds the app for production to the `dist` folder.
